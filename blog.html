Simulating Rigid Body Physics

In this notebook we will take a look at how to implement a physics engine which simulates rigid bodies. We'll explore how you can take physics equations and apply them to moving bodies in discrete time steps, the math behind checking for intersecting objects, and how to generate a collision response.

We will interop Python and C++ so we can use C++ for rendering and Python for the simulation.
